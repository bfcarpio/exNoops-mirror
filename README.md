PR's are welcome!

## Installation

[Available in Hex](https://hex.pm/packages/exnoops), the package can be installed
by adding `exnoops` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:exnoops, "~> 0.1.2"}
  ]
end
```
